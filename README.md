# NexaOS Customize
(this program was made originally by https://github.com/FrancescoCaracciolo) 

Tool to customize NexaOS GNOME appearance easily.

**NexaOS** by **Nexus Foundation** - A modern, beautiful Linux distribution.
## Brand Colors

- **Primary**: #4f46e5 (Indigo)
- **Secondary**: #8b5cf6 (Purple)
- **Background**: #0f121b (Dark Blue)
- **Text**: #f6f6f6 (Light Gray)

## Features

- Easy GNOME desktop customization
- Layout presets for quick setup
- Extension management
- Material You color theming
- Visual effects controls
- System tray and desktop icons management

## Installation

### From Source

```bash
git clone https://github.com/Nexuspenn/NexaCustomize.git
cd NexaCustomize
./install.sh
```

## Usage

Run from your applications menu or terminal:

```bash
nexacustomize
```

## Development

### Requirements

- GNOME 46+
- GTK 4.0
- Python 3.10+
- Meson build system

### Running in Development

```bash
./run.sh
```
